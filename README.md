# bott

Базовый пустой Python-проект, подготовленный для старта и публикации на GitHub.

## Быстрый старт

```powershell
.venv\Scripts\activate
python -m pip --version
```

## Структура

- `.venv/` — локальное виртуальное окружение (в Git не попадает)
- `pyproject.toml` — базовая конфигурация Python-проекта
